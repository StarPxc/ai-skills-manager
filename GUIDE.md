# AI技能管理器 - 用户指南

> 跨平台 AI Skills 本地管理工具，支持浏览、编辑、翻译、禁用/启用、导入导出。

---

## 快速开始

```bash
# 开发运行
npm start

# 打包安装
npm run make       # 当前平台
npm run make:mac   # macOS (.dmg)
npm run make:win   # Windows (.exe)
npm run make:linux # Linux (.deb/.rpm)
```

技能目录默认为 `~/.claude/skills/`，可在设置中自定义。

---

## 功能详解

### 1. 技能浏览与搜索

左侧面板展示所有技能列表，每条显示名称、描述摘要和作者。

- **搜索** — 顶部搜索框实时过滤，支持按名称/描述匹配
- **排序** — 下拉切换排序：更新时间（默认）、名称、作者、版本
- **选中** — 点击条目后右侧展示详情，选中项高亮蓝色左边框
- **禁用** — 右侧 toggle 开关可将技能临时移出扫描目录

> 📸 截图1：主界面全貌，左侧选中一个技能，右侧展示详情、版本/作者标签

### 2. 技能详情与 Markdown 渲染

右侧面板展示技能完整信息：

- 名称、描述、版本号、作者
- **Markdown 渲染** — SKILL.md 内容自动解析为格式化文档（标题、列表、代码块、表格）
- **文本可选可复制** — 支持鼠标选中
- 顶部操作栏：编辑、删除

> 📸 截图2：右侧详情区，展示 Markdown 渲染效果（选一个有表格或代码块的技能）

---

### 3. 编辑技能

点击「编辑」按钮，右侧面板切换为编辑模式：

- **标识** — 技能目录名（只读）
- **显示名称** — 可修改
- **描述** — 多行文本
- **版本 / 作者** — 独立字段
- **触发短语** — 标签式添加/删除
- **SKILL.md 内容** — 等宽字体 textarea 直接编辑
- 保存后即时更新文件，原 frontmatter 格式不丢失

> 📸 截图3：编辑模式，多个表单字段，SKILL.md 编辑器展开

---

### 4. 一键翻译

基于 DeepSeek v4-flash 模型，将技能内容翻译为中文。

- **配置** — 先通过顶部齿轮 → 设置，填入 DeepSeek API Key（在 platform.deepseek.com 获取）
- **翻译** — 点击技能内容区的「译成中文」按钮
- **自动保存** — 译文独立存储为 `SKILL.zh-CN.md`，不修改原文
- **复用** — 下次选中该技能自动加载已有译文，按钮变为「重新翻译」
- **译文复制** — 中文译文区同样支持单独复制

> 📸 截图4：翻译效果，原文 Markdown + 中文译文上下对比

---

### 5. 新建与导入

**新建技能**
- 点击顶部「新增」按钮，输入技能名称即可创建
- 自动生成标准 YAML frontmatter 和 SKILL.md 模板

**导入技能包 (.zip)**
- 点击顶部「导入」，选择 `.zip` 文件
- 自动验证包结构（必须含 SKILL.md）
- 提取技能名、解压到技能目录
- 支持路径穿越防护

> 📸 截图5：导入成功后的提示 Toast

---

### 6. 设置中心

齿轮图标打开设置弹窗：

- **技能目录** — 自定义存储路径，支持浏览选择
- **API Key** — DeepSeek 密钥，密码框输入
- **打开日志** — 一键查看运行日志，排查闪退问题

> 📸 截图6：设置弹窗，展示目录路径、API Key 输入框、打开日志按钮

---

### 7. 深色模式

点击顶部太阳/月亮图标切换，配色遵循系统偏好，状态持久化。

> 📸 截图7：深色模式下的主界面

---

### 8. 日志诊断

应用自动记录运行日志到 `~/.ai-skills-manager.log`：
- 启动参数、平台信息
- 窗口创建、页面加载
- 所有 IPC 操作（增删改查、翻译）
- 渲染进程崩溃详情
- 未捕获异常

日志文件超 2MB 自动轮转。可在设置中一键打开。
